# 🔐 CyberSentinel – AI-Powered Autonomous Threat Hunting System

## 📌 Overview

CyberSentinel is an advanced AI-powered cybersecurity system designed to detect, analyze, and respond to modern cyber threats in real-time. Traditional security systems rely heavily on manual monitoring and predefined rules, making them ineffective against dynamic attacks. CyberSentinel addresses this challenge by providing an intelligent, proactive, and automated threat detection and response platform.

---

## 🚨 Problem Statement

Many organizations face rapidly increasing cyber attacks such as:

* DDoS attacks
* Brute force attacks
* Unauthorized access

Traditional systems are:

* Slow
* Rule-based
* Dependent on manual monitoring

👉 This creates a need for an **intelligent and autonomous threat hunting system**.

---

## 🚀 Proposed Solution

CyberSentinel is an AI-powered autonomous system that:

* Continuously monitors network activity
* Detects abnormal behavior using Machine Learning
* Classifies threats automatically
* Recommends responses using Agentic AI

👉 It reduces human intervention and improves overall security efficiency.

---

## 🔍 Key Features

* 🔍 Real-time network monitoring
  n- 🤖 Automated threat detection using ML
* ⚡ Intelligent response recommendation (Agentic AI)
* 📊 SOC-style interactive dashboard
* 📈 Explainable AI outputs
* 🚨 Live attack feed simulation
* 🔄 Continuous log analysis

---

## 🧠 Working of the System

1. Collects network log data
2. Processes data (packet count, login attempts, etc.)
3. Detects anomalies using ML algorithms
4. Classifies threats into:

   * LOW
   * MEDIUM
   * HIGH
5. Identifies attack types:

   * DDoS
   * Brute Force
6. Agentic AI suggests actions:

   * Blocking suspicious IPs
   * Preventive measures
7. Displays results on dashboard

## ▶️ How to Run

1. Open terminal in project folder
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
python main.py
```
